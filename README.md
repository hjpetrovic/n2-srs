# N2 SRS

Spaced repetition study app for the JLPT N2 exam.

**[Start studying →](https://hjpetrovic.github.io/n2-srs/n2_srs.html)**

## What's inside

- 397 practice questions across 6 types with verified answers and highlighted target words
- 3,024 N2 vocabulary flashcards with readings and English definitions
- 135 grammar patterns with examples
- SM-2 spaced repetition algorithm
- Google sign-in with cloud sync across devices
- Works offline as a PWA — install it on your phone from the browser

## Question types

| Type | Japanese | Count |

## Setup

The app works immediately in guest mode (progress saves locally). For multi-user cloud sync, see [SETUP.md](SETUP.md).

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| `1`–`4` | Select answer / rate card |
| `Space` | Flip flashcard / next card |
| `Enter` | Continue |
